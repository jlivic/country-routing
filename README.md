# country-routing

## Environment setup

Build the project with Maven
  
  mvn clean install
  Start from any tool (preferrably IntelliJ)
  
Test:
  Open terminal and paste:
  
  curl http://localhost:9091/routing/CZE/FRA   - to get 200
  curl http://localhost:9091/routing/CZE/CRO   - to get 400
  
  
Checkout unit tests.
